Como resolver o cubo mágico

Cubo mágico é um puzzle inventado em 1974 pelo professor Húngaro Ernő Rubik.

Para resolver o cubo mágico é antes preciso ter algumas noções básicas, entender a [[Notação do Cubo Mágico]] para saber ler [algoritmos para montagem do cubo mágico] e realizar a [[Memorização de algoritmos]].

Existem diversos métodos para resolução do cubo mágico, Saber resolver o cubo mágico é algo semelhante a fazer mágica, enquanto você não sabe o truque parece algo impossível, mas uma vez que você aprende se torna algo normal, seria algo trivial se não fosse a possibilidade de [[Se tornar mais rápido no 3X3]].

É util ter uma espécie de [[Roadmap cubo mágico]], para saber o que aprender e quais milestones perseguir.

Acredito que a melhor alternativa para se tornar melhor seja a [[A utilização de Prática Deliberada na montagem do cubo mágico]]

Acredito que seja possível usar o [[Anki]], usando [[Repetição Espaçada como forma de aumentar memória de longo prazo]] dos algoritmos aprendidos.
